Sistema Bancário em Python

Projeto simples de um sistema bancário desenvolvido com Python, Flask e SQLite, com interface web e autenticação de usuário.

⸻

 Funcionalidades
	•	Cadastro de usuário
	•	Login
	•	Criar conta
	•	Depositar
	•	Sacar
	•	Visualizar saldo e contas

  Tecnologias
	•	Python
	•	Flask
	•	SQLite
	•	HTML + CSS

  Como executar
  git clone https://github.com/seuusuario/sistema-bancario-python.git
  cd sistema-bancario-python
  pip install -r requirements.txt
  python app.py

  Acesso no navegador
  http://127.0.0.1:5000/login
